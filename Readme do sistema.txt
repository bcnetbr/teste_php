1-Instalar o xampp

2-Executar o script do banco de dados dentro do mysql

3-Colocar a pasta teste dentro da pasta htdocs, localizada na pasta xampp

4-Ajustar as configurações do banco de dados no arquivo db.php localizado dentro da pasta teste

5-inciar o servidor apache através do aplicativo do xampp

6-Digitar no navegador o seguinte endereço: localhost/teste

7-escolher a opção desejada no menu
